CCK PLAN FIELDS
=========================
This module Add a new cck plan field (12 months).

Install
=========================
1) Simply drop this cck_line_product folder into the modules directory (/sites/all/modules/).
2) Install dependences (content, taxonomy)
3) Install via admin/build/modules.



Add cck line products fields
=========================
Add new cck plan field and cck plan field simple like any other cck field.
